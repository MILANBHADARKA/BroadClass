# Formatting Compliance Checklist (DDU IT Report)

Use this checklist while converting the draft into final Word/PDF submission format.

## 1. Page Layout and Numbering

- Cover page and first page: no page numbers.
- Candidate declaration to pre-chapter pages: small roman numerals (i, ii, iii...).
- Chapter 1 onward: Arabic numerals (1, 2, 3...).
- Bottom-right: page number.
- Bottom-left: DDU(Faculty of Tech., Dept. of IT).
- Top-right: chapter heading.

## 2. Fonts and Headings

- Chapter heading: Times New Roman, 16 pt, bold, ALL CAPS.
- Section heading: Times New Roman, 14 pt, bold, ALL CAPS.
- Subsection heading: Times New Roman, 12 pt, bold, Leading Caps.
- Body text: Times New Roman, 12 pt.
- Special text (equations/footnotes): Times New Roman Italic, 12 pt.

## 3. Spacing and Alignment

- Single line spacing in body text.
- Double spacing between paragraphs.
- Double spacing between chapter and section headings.
- Double spacing between section heading and paragraph.
- Fully justified paragraphs.
- Bullets/list text can remain indented and single-spaced.

## 4. Figures and Tables

- Figure caption at bottom: Fig <chapter>.<serial> <title>.
- Table caption at top: Table <chapter>.<serial> <title>.
- Figures/tables must appear immediately after first reference.
- Landscape items rotated with top toward left.
- Ensure all figures and tables are center aligned.

## 5. Chapter Structure Validation

- Verify all required chapters 1 through 10 are present.
- Verify all required subsection numbering from institute template is preserved.
- Ensure appendices and references are included.
- Add personal sections later: declaration, certificate, acknowledgement, experience.

## 6. Final Submission Artifacts

- Add source code and executable package to guide copy media.
- Attach report soft copy.
- Ensure number of spiral-bound copies matches group size + guide copy.

## 7. Final Quality Review

- Grammar and consistency check.
- Reference formatting check (alphabetical and numbered).
- Cross-check all in-text references to figures/tables.
- Replace screenshot placeholders with actual UI captures.
